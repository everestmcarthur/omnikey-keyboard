### Key Components

#### OmniKeyService
Main `InputMethodService` that:
- Manages keyboard lifecycle
- Handles key presses, swipes, gestures
- Routes input to appropriate engines
- Detects input context (terminal, code, writing)
- Applies app-specific settings

#### PredictionEngine
- N-gram word prediction model
- Learns from typing history (Room database)
- Prefix-based completion
- Context-aware suggestions

#### CodeEngine
- Language detection (simple heuristics)
- Snippet expansion with placeholders
- Auto-pairing: brackets, quotes
- Terminal command completion
- Undo/redo stack (50 levels)

#### WriterEngine
- Grammar rules (regex-based)
- Readability scoring (Flesch-Kincaid)
- Smart punctuation (curly quotes, em-dashes)
- Markdown auto-completion

#### ThemeEngine
- 10 built-in themes
- Unlimited custom themes (stored in Room)
- RGB effects (would require custom view rendering)
- Background image support

## рџљЂ Getting Started

### Prerequisites
- Android Studio Hedgehog (2023.1.1) or later
- JDK 17
- Android SDK 34
- Gradle 8.2+

### Build & Install

```bash
# Clone the repository
git clone https://github.com/yourusername/omnikey.git
cd omnikey

# Build debug APK
./gradlew assembleDebug

# Install to device
./gradlew installDebug
```

### Enable Keyboard

1. Open **OmniKey** app
2. Tap **Enable Keyboard** в†’ Goes to System Settings
3. Toggle **OmniKey** on
4. Tap **Select Keyboard** в†’ Choose OmniKey
